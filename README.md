# PassWordManager
密码加密记录备份管理工具

## 使用
1. 打开/创建 按钮: 存在的文件则打开，不存在的则创建
2. 确认文件密码: 文件存在密码则确认密码，否则（新创建的文件）为设置密码
3. 修改按钮: 文件成功打开后方可修改文件密码
4. 在下方添加区添加项，其中关键字、备注支持中文但不支持中文字符
5. 查询点击下拉展示关键字，输入则匹配关键字、用户名、密码进行查询
6. 点击表头 关键字、用户名、备注可对查询结果进行排序显示
